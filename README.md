# TaskMate

A simple task management application for Android.

## Features

*   Add, edit, and delete tasks.
*   Mark tasks as complete.
*   Sort tasks by title or due date.
*   Search for tasks.
*   Undo task deletion.

## How to Build and Run

### Prerequisites

*   [Android Studio](https://developer.android.com/studio) installed on your machine.
*   An Android emulator or a physical Android device.

### Steps

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/thr-tanvir/TaskMateApp
    ```

2.  **Open the project in Android Studio:**
    *   Open Android Studio.
    *   Select **File > Open** (or Open an Existing Project on the welcome screen).
    *   Navigate to the directory where you cloned the repository and select it.
    *   Android Studio will automatically sync the project with Gradle.

3.  **Run the application:**
    *   Make sure you have an Android emulator set up or a physical device connected to your computer.
    *   Select the device from the dropdown menu in the toolbar.
    *   Click the **Run 'app'** button (the green triangle icon) or select **Run > Run 'app'** from the menu.

# The application will be built, installed, and launched on the selected device/emulator.
